创建数据库

Intellij 自动生成 Model

  1. 右键项目，选择Add Framework Support
  2. 下拉选择JavaEE Persistence，右边 provider 选择Hibernate
  3. 此时resources里面生成了persistence.xml配置文件。同时左下角出现了persistence边框
  4. 右键项目名，选择Generate Persistence Mapping，再选择By Database Schema
  6. 打开model包，可以看到生成了两个Java Bean，在SpringMVC中称为两个实体，它们对应了数据库的两张表。
  7. 将persistence.xml文件移除（过于陈旧）。把数据库配置移到mvc-dispatcher-servlet.xml中。
  • jpa:repositories：这一部分涉及到数据库的接口
  • entityManagerFactory：实体管理器工厂，读取persistence.xml配置
  • transactionManager：事务管理器，利用entityManager进行事务管理
  • tx:annotation-driven：打开事务管理器的注解驱动，可以使用注解的方法操纵数据库。
